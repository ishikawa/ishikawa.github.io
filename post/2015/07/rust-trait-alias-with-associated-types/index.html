Aliasing trait with Associated Types

2015/07/26 10:10am

Rust の型には type キーワードによる別名づけの手段が用意されているが、これは Traits には使えない。

trait Trait {}
struct Struct { foo: i32 }

// ERROR: note: `type` aliases cannot be used for traits
type Alias = Trait;
impl Alias for Struct {}

には、通常の型のように type による

trait CharIterator: Iterator<Item=char> {}
impl<T: Iterator<Item=char>> CharIterator for T {}
