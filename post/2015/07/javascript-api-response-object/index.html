JavaScript API Response object

2015/07/07 8:02am

JavaScript (TypeScript) でサーバーとの通信には JSON-RPC 2.0 を用いて、受け取った JSON レスポンスをオブジェクトに変換して使っている。

class Tweet {

  constructor(
    userId: string,
    screenName: string,
    text: string,
    favorited: boolean,
  ) {}

  static fromJsonResponse(response: any): Tweet {
    return new Tweet(
      response["userId"],
      response["screenName"],
      response["text"],
      response["favorited"]
    );
  }
}

TypeScript のクラスはインターフェースとしても使える ので、JSON レスポンスを変換したオブジェクトをそのまま使える気がしてきた。


class Tweet {
    userId: string;
    screenName: string;
    text: string;
    favorited: boolean;
