Apache XML-RPC のファイル名

2006/12/10 1:04am

Apache XML-RPC で 配布されている .jar は依存ライブラリを除くと 3 つあるのですが、

  • xmlrpc-client-3.0.jar
  • xmlrpc-common-3.0.jar
  • xmlrpc-server-3.0.jar

ファイル名を Mac の Finder からエディタにコピペして気づいた。

xmlrpc-client-3.0.jar
xmlrpc-common-3.0.jar
xmlrpc-server-3.0.jar

すべて文字数が一緒なんです。
