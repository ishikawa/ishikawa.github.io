ffmpeg を MacPorts でインストール

2007/12/09 9:57am

  • mac

動画の変換がやりたくて Mac に ffmpeg をインストールした。そのときのメモ。

MacPorts でインストールするのがよさげ。variants のおかげで、H.264 や AAC を有効にするのも簡単だ。

% sudo port install x264
% sudo port install faac
% sudo port install faad2
% sudo port install a52dec
% sudo port install ffmpeg +faac +faad +a52 +x264

関連するライブラリを先にインストールして、+faac のように指定してやるだけ。
