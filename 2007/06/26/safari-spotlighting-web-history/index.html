Safari の履歴検索がすごく便利です

2007/06/27 8:06am

上のスクリーンショット（クリックすると拡大した画像に飛びます）を見れば、一目瞭然。Safari は履歴も検索できる。

これで、「あー、あの Web ページ、はてブか何かで見たんだけど、どこだっけなー」なんて悩みともおさらばできます。て、普通にみんな使ってるんだろうか、この履歴検索。

検索窓が、普段意識しない場所（ブックマークウインドウの下側）にあるものだから、いまのいままで気づかなかったよ。

  • Rails の便利メソッド h, html_escape は ERB:Util にあった
  • Ruby の Time.parse で文字列を Time に変換するときのエラーチェック
  • Twitter
  • Github
